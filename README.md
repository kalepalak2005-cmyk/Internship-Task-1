# Customer Behavior Analysis

## Overview

This repository contains an end-to-end customer behavior analysis project completed as part of an internship task. The objective is to analyze customer transaction and behavior data to identify meaningful customer segments, understand purchase patterns, assess churn risk, and translate insights into actionable business recommendations.

The analysis is based on a real-world Kaggle dataset and focuses on both analytical rigor and business relevance.

## Objectives

* Clean and preprocess raw transactional data
* Engineer relevant behavioral features
* Segment customers using RFM analysis and/or clustering techniques
* Analyze purchase behavior and retention trends
* Identify potential churn-risk segments
* Provide data-driven recommendations to improve customer engagement

## Methodology

1. **Data Cleaning & Preprocessing**

   * Handling missing values and duplicates
   * Data type corrections and consistency checks

2. **Feature Engineering**

   * Creation of Recency, Frequency, and Monetary (RFM) metrics
   * Additional behavioral indicators where applicable

3. **Customer Segmentation**

   * RFM-based segmentation
   * Clustering techniques for customer grouping

4. **Exploratory Data Analysis & Visualization**

   * Purchase pattern analysis
   * Retention and engagement trends
   * Segment-wise behavior comparison

5. **Insights & Recommendations**

   * Interpretation of customer segments
   * Identification of high-value and churn-prone customers
   * Business-focused recommendations

## Deliverables

* **Jupyter Notebook**: Complete analysis with code and visualizations
* **PDF Report**: Summary of customer segments, key findings, and actionable recommendations
